# 关于PhotoCollection
----------

## 简介


- 简单的Flickr网站的自定义客户端，它能获取并展示Flickr网站的最新公共图片。

- 支持关键词搜索。

- 支持后台定时查询最新图片信息并提供通知。

- 支持应用内网页浏览。
